# Quoi de neuf dans Balise 0.24.0

### Tags magiques configurables
Les **tags magiques** — les tags assignés automatiquement à vos notes selon leur contenu — sont désormais entièrement configurables dans **Paramètres > Tags magiques**.

Chaque règle associe un motif à un tag, avec quatre modes de correspondance : **Commence par**, **Se termine par**, **Contient** et **Contient le mot** (séparé par des espaces). Les règles par défaut (`- [ ]` → `todo` et `- [x]` → `done`) restent en place, et vous pouvez en ajouter, modifier ou supprimer autant que vous souhaitez.

Un aperçu s'affiche sous chaque règle pour visualiser comment le motif sera reconnu selon le mode choisi.
