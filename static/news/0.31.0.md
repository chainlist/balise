# Quoi de neuf dans Balise 0.31.0

### Sommaire de la note
Un nouveau bouton en haut à droite de l'éditeur ouvre le **Sommaire** : la liste de tous les titres de la note, hiérarchisés selon leur niveau. Cliquez sur un titre pour vous y rendre directement, sans faire défiler. Pratique pour se repérer dans les notes longues.


### Naviguer entre les occurrences d'un tag
Les tags de la note s'affichent désormais en haut de l'éditeur. Cliquez sur l'un d'eux pour faire apparaître un petit navigateur : il indique le **nombre d'occurrences** du tag dans la note et permet de **passer de l'une à l'autre** avec les flèches haut et bas. Idéal pour retrouver tous les endroits où un sujet est mentionné.


### Insérer des dates dans vos notes
Balise sait désormais insérer des dates, de plusieurs façons :
- Tapez `@` dans l'éditeur pour faire apparaître un **calendrier** et choisir la date de votre choix.
- Utilisez le raccourci `Ctrl + ;` (ou `Cmd + ;` sur Mac) pour insérer **la date du jour** à la position du curseur.
- Depuis le menu des commandes « / », retrouvez **Date du jour** et **Insérer une date**.

Le **format de date** se règle dans **Paramètres > Général** et s'adapte à la langue de l'application.


### Paramètres réorganisés
Les sections des paramètres sont désormais regroupées pour s'y retrouver plus facilement. Les **tags** disposent de leur propre section, dans le groupe **Organisation**, où vous pouvez les gérer et les personnaliser.


### Améliorations et corrections
- Dans la barre latérale, **Tâches** passe désormais avant **Toutes les notes**.
- Correction de la fermeture automatique des **blocs de code** (```).
- Ajustement de l'arrondi des fenêtres d'edition de l'éditeur.
- Correction de l'affichage du **séparateur** dans l'éditeur.
- Les panneaux latéraux givrés (effet de flou) s'affichent de nouveau correctement.
