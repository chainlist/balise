# What's new in Balise 0.18.0

### Knowledge graph
Visualise how your tags connect to each other with the new graph view, available from the sidebar where the dashboard used to be. Drag nodes around, pan, zoom, click a tag to highlight its neighbours, double-click to jump to that tag's notes. A closable settings panel in the top-right lets you filter by minimum connection strength, hide isolated tags, and tune the force layout.

### Foldable sections in the editor
Long notes can now be collapsed in the editor. Use the fold gutter on the left edge to toggle headings, lists, and code blocks.

### Coloured headings
Headings now use distinct colours so document structure is easier to scan at a glance.

### Tab to nest list items
Pressing Tab inside a list item now nests it; Shift-Tab unnests. Matches the behaviour from most other note apps.

### Smarter autocomplete and trim on save
Tag autocomplete now responds to the standard editor keymap. Notes are also stripped of trailing whitespace each time they're saved, so files stay clean on disk.

### Resizable notes panel
The notes list between the sidebar and the editor can now be resized by dragging its right edge. Your chosen width is remembered across sessions.
