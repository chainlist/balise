# Quoi de neuf dans Balise 0.30.0

### Sommaire de la note
Un nouveau bouton en haut à droite de l'éditeur ouvre le **Sommaire** : la liste de tous les titres de la note, hiérarchisés selon leur niveau. Cliquez sur un titre pour vous y rendre directement, sans faire défiler. Pratique pour se repérer dans les notes longues.


### Naviguer entre les occurrences d'un tag
Les tags de la note s'affichent désormais en haut de l'éditeur. Cliquez sur l'un d'eux pour faire apparaître un petit navigateur : il indique le **nombre d'occurrences** du tag dans la note et permet de **passer de l'une à l'autre** avec les flèches haut et bas. Idéal pour retrouver tous les endroits où un sujet est mentionné.


### Améliorations et corrections
- Correction de l'affichage du **séparateur** dans l'éditeur.
- Les panneaux latéraux givrés (effet de flou) s'affichent de nouveau correctement.
