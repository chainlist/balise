# Quoi de neuf dans Balise 0.23.0

### Mode graphe en réseau
La vue graphe propose un nouveau mode **Réseau** en complément de la vue **Radiale** existante. Au lieu d'un anneau unique, chaque tag devient un nœud et chaque co-occurrence devient un lien, disposés par simulation de forces afin que les tags liés se regroupent naturellement.

Vous pouvez déplacer les nœuds, faire défiler pour zoomer et faire glisser l'arrière-plan pour naviguer. Survoler un tag le met en évidence avec ses connexions directes tout en estompant le reste. Cliquer sur un tag le fixe au centre comme point focal ; double-cliquer ouvre les notes de ce tag.


### Capture rapide
Une nouvelle fenêtre de **capture rapide** permet d'ajouter une note sans quitter votre travail en cours. Ouvrez-la depuis n'importe quelle application avec le raccourci global **Ctrl+Maj+Espace**, une fenêtre épurée s'affiche avec uniquement l'éditeur, prête à saisir.

La note n'est enregistrée qu'au premier caractère saisi. Si vous fermez la fenêtre sans rien écrire, aucune note vide n'est créée. Vous pouvez également épingler la fenêtre au premier plan avec le bouton punaise, pratique pour garder la fenetre toujours visible pendant que vous travaillez.
