# Quoi de neuf dans Balise 0.28.0

### Synchronisation entre vos appareils
Grosse nouveauté : Balise peut maintenant **synchroniser vos notes entre plusieurs appareils**. Tout se passe dans le nouvel onglet **Paramètres > Synchronisation**.

Pour associer deux appareils, affichez un code sur l'un et saisissez-le sur l'autre. Une fois jumelés, vos appareils apparaissent dans **Appareils liés**, où vous pouvez les renommer, les dissocier ou lancer une synchronisation à la demande. Vous pouvez aussi régler la **fréquence de synchronisation** selon vos préférences.

Vous gardez la main sur ce qui est partagé dans l'onglet **Partage** : choisissez de synchroniser ou non vos paramètres, et désactivez un bureau précis pour le garder uniquement sur l'appareil courant. Les utilisateurs avertis trouveront aussi des **paramètres avancés** pour pointer vers leur propre serveur de jumelage.


### Mise à jour automatique des notes modifiées en dehors de l'app
Si vous éditez un fichier `.md` directement sur le disque, ou qu'une note arrive via la synchronisation, Balise le détecte et **recharge les notes concernées** automatiquement. Une petite notification vous indique combien de notes ont été mises à jour.


### Citations dans l'éditeur
Commencez une ligne par `>` pour créer une **citation**. Une barre verticale apparaît sur la gauche pour bien la distinguer du reste du texte. Une ligne de citation vide se referme proprement d'une simple pression sur Entrée.


### Blocs de code repensés
Les **blocs de code** ont été entièrement refaits. Ils restent désormais modifiables directement, comme le reste de votre note, avec une coloration syntaxique selon le langage indiqué après les trois accents graves. Au passage, l'application s'est allégée d'une dépendance, ce qui la rend un peu plus rapide.


### Tags magiques et page À propos remis au goût du jour
Le section **Tags magiques** a été repensé pour être plus clair et plus agréable à utiliser. La section **À propos** a elle aussi été retravaillée : numéro de version, vérification des mises à jour et remerciements aux outils qui font tourner Balise.


### Améliorations et corrections
- La barre latérale et la navigation des paramètres ont été repensées pour s'y retrouver plus facilement.
- Dans la fenêtre d'ajout rapide, vous pouvez maintenant changer de bureau sans la fermer.
- Les réglages de taille de police et d'interligne de l'éditeur utilisent un sélecteur plus pratique.
- Correction d'un souci d'affichage des menus déroulants qui ne s'ouvraient pas correctement.
