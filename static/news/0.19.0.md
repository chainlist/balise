# What's new in Balise 0.19.0

### Tasks board
A new Tasks view is available from the sidebar. It aggregates every task across all your notes into a three-column kanban board — **To Do**, **In Progress**, and **Done** — so you can see everything at a glance without opening individual notes.

Tasks are picked up automatically from two syntaxes already supported in the editor: status hashtags (`#todo`, `#inprogress`, `#done`) on a line, and Markdown checklists (`- [ ]`, `- [x]`). Drag a card between columns to update its status directly in the source note — the underlying markdown is rewritten and synced to disk as usual. Clicking a card navigates to the parent note in the editor. Checklist cards are visually distinct (dashed border, checkbox glyph) and can only move between To Do and Done, since the checkbox syntax has no in-progress form.

### German language support
Balise is now fully translated into German. Switch to Deutsch from Settings → Language.
