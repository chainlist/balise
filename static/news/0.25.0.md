# Quoi de neuf dans Balise 0.25.0

### Couleur principale personnalisable
La **couleur principale** de l'interface, utilisée pour les boutons, les liens et les surlignages, est désormais personnalisable dans **Paramètres > Apparence**. Choisissez une teinte parmi la palette dédiée : l'application adapte automatiquement les nuances pour conserver le même contraste en thème clair comme en thème sombre. Un clic sur **Réinitialiser** restaure la couleur d'origine.


### Arrière-plan : couleur unique et taille des bulles
L'arrière-plan coloré gagne deux nouveautés dans **Paramètres > Apparence** :

- Un nouveau mode **Couleur unique** remplace les quatre coins par une seule couleur centrale, pour un fond plus uniforme.
- En mode **Coins**, chaque bulle de couleur dispose maintenant d'un réglage de **taille** (de 50 % à 200 %), accessible directement dans le sélecteur de couleur.

Les valeurs par défaut ont également été revues pour un rendu plus doux.


### Description d'images
Double-cliquez sur une image dans l'éditeur, ou survolez-la et cliquez sur le crayon, pour lui ajouter une **description**. Elle s'affiche discrètement sous l'image et est enregistrée dans le texte de la note comme texte alternatif.


### Menu de commandes repensé
Le menu « / » de l'éditeur a été repensé : il reste désormais toujours visible à l'écran (il se repositionne près du bord de la fenêtre), devient défilable quand la liste est longue, et l'élément sélectionné au clavier reste toujours en vue.


### Améliorations et corrections
- Le sélecteur de thème (clair / sombre / système) adopte un design plus compact.
- Le calendrier du journal s'ouvre désormais dans un menu givré, cohérent avec le reste de l'interface.
- Les liens dans l'aperçu des notes s'ouvrent dans votre navigateur par défaut au lieu de naviguer dans l'application.
- La fenêtre d'ajout rapide suit désormais les changements de thème clair/sombre.
- L'espacement des tags dans la barre latérale a été ajusté.
