# What's new in Balise 0.22.0

### Network graph mode
The graph view has a new **Network** mode alongside the existing **Radial** view. Instead of a single ring, every tag becomes a node and every co-occurrence becomes a link, laid out by a force simulation so related tags naturally cluster together.

You can drag nodes around, scroll to zoom, and drag the background to pan. Hovering a tag highlights it and its direct connections while dimming the rest. Clicking a tag pins it to the center as the focal hub; double-clicking opens that tag's notes.

### Network view settings
The settings panel in Network mode adds controls tuned for the force layout: **Spread** (how strongly tags push apart), **Link length** (the resting distance between connected tags), and a **Hide isolated tags** option to drop tags that have no connections.

### Remembered graph mode
Balise now remembers whether you last used the Radial or Network view and reopens the graph in that mode on the next launch.
