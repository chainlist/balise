# What's new in Balise 0.16.0

## Balise now loads 10x faster
Boot time has been significantly reduced through backend optimizations. The app should feel snappier from the moment you open it.

### Desk management
You can now rename or delete desks directly from the new desk settings modal. Click the settings icon next to any desk name in the sidebar to get started.

### Search in shortcuts
The shortcuts panel in Settings now includes a search bar, making it easy to find and reassign any shortcut without scrolling through the full list.

### What's new dialog
Balise now shows a release notes dialog when you update to a new version. You can also re-read the latest notes anytime from Settings > About.

### Fix: note titles being cut off
Long note titles in the sidebar were not truncating correctly. This is now fixed.
