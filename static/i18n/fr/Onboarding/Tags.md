# Les Tags

Les tags sont le principal moyen d'organiser vos notes dans Balise. Ecrivez un `#hashtag` n'importe ou dans une note et il devient automatiquement un tag de cette note.

```
Lu un super livre aujourd'hui. #lecture #2024
```

Les tags apparaissent dans la barre laterale sous forme de liste. Cliquer sur un tag filtre vos notes pour n'afficher que celles qui le contiennent, comme un dossier dynamique.

Quelques points importants :

- Les tags sont extraits en temps reel pendant la frappe
- Une note peut avoir autant de tags que vous le souhaitez
- Les tags fonctionnent pour toutes les notes d'un bureau
- Tapez `#` dans l'editeur pour obtenir des suggestions d'autocompletion parmi vos tags existants

Comme les tags sont dans le contenu de la note, ils suivent vos fichiers si vous les deplacez.
