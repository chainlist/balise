# Editeur et Markdown

Balise utilise un editeur en texte brut avec support du markdown. Voici un guide de reference rapide :

## Titres

```
# Titre 1
## Titre 2
### Titre 3
```

## Mise en forme

```
**gras**
*italique*
~~barre~~
```

## Listes

```
- Element un
- Element deux
  - Element imbrique

1. Premier
2. Deuxieme
```

## Liens et images

```
[texte du lien](https://exemple.com)
![texte alternatif](image.png)
```

## Code

```
Code `inline` ou bloc delimite :

    ```
    const x = 1
    ```
```

## Citations

```
> Un passage cite
```

## Separateur horizontal

```
---
```

L'editeur supporte la fermeture automatique des parentheses et guillemets. Utilisez `Ctrl+B` pour le gras et `Ctrl+I` pour l'italique.
