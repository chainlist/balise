# Bienvenue sur Balise

Balise est une application de prise de notes en markdown qui tourne sur votre bureau. Vos notes restent en local, dans des fichiers `.md` lisibles et modifiables avec n'importe quel outil.

Quelques raccourcis pour bien commencer :

- **Creer une note** avec `Ctrl+N`
- **Rechercher** avec `Ctrl+K`
- **Changer de bureau** depuis la barre laterale

Les notes dans ce dossier vous presentent les concepts cles. Vous pouvez les supprimer une fois que vous etes a l'aise.

Bonne ecriture !
