# Tags

Tags are the main way to organize notes in Balise. Write a `#hashtag` anywhere in a note and it becomes part of that note's tags automatically.

```
Picked up a great book today. #reading #2024
```

Tags appear in the sidebar as a list. Clicking one filters your notes to show only notes that contain that tag, acting like a dynamic folder.

A few things to know:

- Tags are extracted live as you type
- A note can have as many tags as you like
- Tags work across all notes in a desk
- Type `#` in the editor to get autocomplete suggestions from your existing tags

Because tags live inside the note content, they travel with your files if you ever move them.
