# Editor and Markdown

Balise uses a plain text editor with markdown support. Here is a quick reference:

## Headings

```
# Heading 1
## Heading 2
### Heading 3
```

## Emphasis

```
**bold**
*italic*
~~strikethrough~~
```

## Lists

```
- Item one
- Item two
  - Nested item

1. First
2. Second
```

## Links and Images

```
[link text](https://example.com)
![alt text](image.png)
```

## Code

```
Inline `code` or a fenced block:

    ```
    const x = 1
    ```
```

## Blockquotes

```
> A quoted passage
```

## Horizontal rule

```
---
```

The editor supports auto-closing brackets and pairs. Use `Ctrl+B` for bold and `Ctrl+I` for italic.
