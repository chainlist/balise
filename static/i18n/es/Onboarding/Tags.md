# Tags

Los tags son la principal forma de organizar notas en Balise. Escribe un `#hashtag` en cualquier parte de una nota y se convierte automaticamente en un tag de esa nota.

```
Hoy empece un libro genial. #lectura #2024
```

Los tags aparecen en la barra lateral como una lista. Al hacer clic en uno, se filtran las notas para mostrar solo las que lo contienen, funcionando como una carpeta dinamica.

Algunos puntos importantes:

- Los tags se extraen en tiempo real mientras escribes
- Una nota puede tener tantos tags como quieras
- Los tags funcionan para todas las notas de un escritorio
- Escribe `#` en el editor para obtener sugerencias de autocompletado entre tus tags existentes

Como los tags viven dentro del contenido de la nota, se mueven con tus archivos si los copias o trasladas.
