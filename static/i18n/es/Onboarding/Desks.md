# Los Escritorios

Un **escritorio** es un espacio de trabajo aislado. Cada escritorio tiene su propio conjunto de notas, etiquetas y una carpeta dedicada en tu sistema de archivos.

Usa los escritorios para separar areas de tu vida o trabajo, por ejemplo:

- Un escritorio para un diario personal
- Un escritorio para un proyecto profesional
- Un escritorio para notas de investigacion o lectura

Cada escritorio guarda sus notas en `~/Documents/Balise/{nombre del escritorio}/`. Los archivos son markdown puro, portables y legibles fuera de Balise.

Cambia de escritorio desde la barra lateral. Crea un nuevo escritorio desde el panel de configuracion.
