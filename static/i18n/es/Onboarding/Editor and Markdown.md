# Editor y Markdown

Balise usa un editor de texto plano con soporte de markdown. Aqui tienes una referencia rapida:

## Encabezados

```
# Encabezado 1
## Encabezado 2
### Encabezado 3
```

## Enfasis

```
**negrita**
*cursiva*
~~tachado~~
```

## Listas

```
- Elemento uno
- Elemento dos
  - Elemento anidado

1. Primero
2. Segundo
```

## Enlaces e imagenes

```
[texto del enlace](https://ejemplo.com)
![texto alternativo](imagen.png)
```

## Codigo

```
Codigo `inline` o bloque delimitado:

    ```
    const x = 1
    ```
```

## Citas

```
> Un parrafo citado
```

## Linea horizontal

```
---
```

El editor cierra automaticamente los parentesis y comillas. Usa `Ctrl+B` para negrita y `Ctrl+I` para cursiva.
