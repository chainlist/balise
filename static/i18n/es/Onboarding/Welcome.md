# Bienvenido a Balise

Balise es una aplicacion de notas en markdown que funciona en tu escritorio. Tus notas se guardan localmente en archivos `.md` que puedes leer y editar con cualquier herramienta.

Algunos atajos para empezar:

- **Crear una nota** con `Ctrl+N`
- **Buscar notas** con `Ctrl+K`
- **Cambiar de escritorio** desde la barra lateral

Las notas de esta carpeta te presentan los conceptos principales. Puedes eliminarlas cuando ya te sientas comodo.

Buena escritura!
